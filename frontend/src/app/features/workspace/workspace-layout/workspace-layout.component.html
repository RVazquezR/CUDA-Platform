El Sistema de Tokens y Colas

Tu posición en la cola de procesamiento depende de tus Tokens. Si posees tokens, tu código entrará en la vía rápida. Si agotas tus tokens, tu ejecución se colocará al final de la cola. Los tokens se renuevan periódicamente. Nota: El Administrador tiene prioridad absoluta.

Recuperación de Sesión

Si envías un código y cambias de pestaña, se guardará tu sesión. Si el programa sigue en espera o ejecución a tu regreso, la terminal se reconectará automáticamente a la tarea.

Concurrencia y Timeout

El administrador establece la concurrencia máxima permitida y el tiempo límite. Si tu código supera el tiempo permitido, será abortado automáticamente.

Restauración Gratuita

Si la ejecución finalizó mientras estabas en otra ventana, usa el botón "Restaurar Última Ejecución" para ver la salida por consola sin gastar tokens.

Abortar Ejecución

Si pulsas el botón rojo para abortar un proceso en curso, el sistema detendrá la tarea inmediatamente y te reembolsará el token utilizado.

Precaución: Límite de Cuota Máxima de Almacenamiento

Si tu código genera un archivo de salida (ej. resultado.bmp) y junto a todos tus archivos se excede tu cuota de almacenamiento disponible, el archivo será descartado permanentemente por el sistema para proteger el disco del servidor. Consulta tu cuota en el apartado "Mis Métricas".

  • Llamadas en el código: Cuando escribas código CUDA, llama a los archivos exactamente con el nombre que ves en la lista de la derecha (ej. fopen("imagen.bmp", "r");).
  • Archivos Globales vs Personales: Los archivos globales los proporciona el administrador y no consumen tu cuota. Además, si subes un archivo personal que se llama exactamente igual que uno global, tu código CUDA siempre usará el tuyo.
  • Sobrescritura y Nombres: El sistema de subida no sobrescribe. Si subes dos veces el mismo archivo, verás duplicados en la lista. Para evitar que tu código se confunda y procese el archivo incorrecto, renómbralos en tu ordenador antes de subirlos. Asegúrate también de que tu código genere archivos de salida con nombres únicos.
  • Gestión de Interfaz: Pasa el ratón por encima de cualquier archivo personal en la lista para revelar los botones rápidos de Descarga y Eliminación.
