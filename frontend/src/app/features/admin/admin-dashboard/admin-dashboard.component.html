Usa estos controles solo si el servidor está sobrecargado o detectas usos maliciosos.

Congelar GPU: Detiene el procesamiento temporalmente. Las tareas se acumularán de forma segura en la cola de espera ("Pending").

Activar Motor: Vuelve a encender la GPU. Las tareas acumuladas comenzarán a procesarse inmediatamente según su orden y tokens.

Eliminación Total: Cancela y elimina todas las tareas en espera o procesándose. Devuelve los tokens a los alumnos afectados automáticamente.
